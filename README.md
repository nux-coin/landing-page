# Nux Coin Landing Page

Coming-soon landing page for **Nux Coin**, an ERC-20 token minting on **Polygon**, built with Next.js.

## Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS v4

## Run Locally

```bash
pnpm install
pnpm dev
```

Open `http://localhost:3000`.

## Production Build

```bash
pnpm build
pnpm start
```

## Structure

- `src/app/page.tsx` — main landing page
- `public/nux-coin.svg` — logo asset
- `public/banner.svg` — hero banner asset

## Quick Customization

- Update headline, token details, and roadmap in `src/app/page.tsx`
- Replace brand files in `public/` when needed
- Update social links in the CTA section
